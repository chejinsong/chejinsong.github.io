第一章 用Three.js创建第一个三维场景

前段时间看了一些webgl的页面效果，被他的强大的渲染技术给惊呆了，于是在网上找资料学习，但是不像其他的框架，webgl资料好少，除了github上的，别的都很少，有个中文网也是需要付费才能学习，但是，身为一个屌丝，哪来的那么多钱呢(自己研究记忆更深刻！)。以后就是学一小节更新一次，做个笔记吧，不然到最后学到底，什么东东都没保留就杯具了。G O →

准备工作

  1. 编辑器（编辑器可以根据自己的喜好选择使用，这个没有强制性的要求）
  2. 教程源代码(点击下载)
  3. 测试环境（在本地开启一个服务，如果直接打开html也页面，有些测试用例需要加载外部资源，比如三维模型和图片时候，仅仅打开html页面是不行的，这个时候我们就需要建立一个本地web服务器，方便我们测试，任何服务都可以，自选吧，我自己使用的node，一个字:方便）

先看看教材上面的例子(文件是02-first-scene.html)：

<!DOCTYPE html>
<html>
<head>
    <title>Example 01.01 - Basic skeleton</title>
<script type="text/javascript" src="../learning-threejs-master/libs/three.js"></script>
<style>
    body {
        margin: 0;
        overflow: hidden;
    }
</style>
</head>
<body>
  <div id="WebGL-output"></div>
<script type="text/javascript">
function init() {
    var scene=new THREE.Scene();
    var camera=new THREE.PerspectiveCamera(45,window.innerWidth/window.innerHeight,0.1,1000);
    var renderer=new THREE.WebGLRenderer();
    // 设置渲染器清除颜色
    renderer.setClearColorHex(0xeeeeee);
    // 设置渲染器大小
    renderer.setSize(window.innerWidth,window.innerHeight);
    // 
    var axes=new THREE.AxisHelper(20);
    scene.add(axes);

    var planeGeometry=new THREE.PlaneGeometry(60,20,1,1);
    var planeMaterial=new THREE.MeshBasicMaterial({color:0xcccccc});
    var plane=new THREE.Mesh(planeGeometry,planeMaterial);
    plane.rotation.x=-0.5*Math.PI;

    plane.position.x=15;
    plane.position.y=0;
    plane.position.z=0;

    scene.add(plane);

    var sphereGeometry=new THREE.SphereGeometry(4,20,20);
    var sphereMaterial=new THREE.MeshBasicMaterial({color:0x7777ff,wireframe:true});
    var sphere=new THREE.Mesh(sphereGeometry,sphereMaterial);
    sphere.position.x=20;
    sphere.position.y=4;
    sphere.position.z=2;

    scene.add(sphere);

    camera.position.x=-30;
    camera.position.y=40;
    camera.position.z=30;
    camera.lookAt(scene.position);

    document.querySelector('#WebGL-output').append(renderer.domElement);
    renderer.render(scene,camera);
}
window.onload = init
</script>
</body>
</html>

源码解析

在服务器中打开页面，就可以看到场景中放着面板小球和坐标，好 我们先来解释一下代码，这样你就知道在干嘛了：

var scene=new THREE.Scene();
   var camera=new THREE.PerspectiveCamera(45,window.innerWidth/window.innerHeight,0.1,1000);
   var renderer=new THREE.WebGLRenderer();
   // 设置渲染器清除颜色
   renderer.setClearColorHex(0xeeeeee);
   // 设置渲染器大小
   renderer.setSize(window.innerWidth,window.innerHeight);

事例一开始定义了场景(scene),相机(camera),还有渲染器(renderer).
变量scene是一个容器，用来保存并且跟踪我们想要渲染的物体，我们想要渲染的球和方块后面会添加到scene中，这个例子中还创建了camera，camera变量定义了我们在场景(scene)中可以看到什么。接下来又创建了renderer对象，它负责计算指定相机角度下的场景的样子。我们在代码里通过renderer.setClearColorHex(0xeeeeee)来给渲染器设置颜色，并且通过renderer.setSize(window.innerWidth,window.innerHeight);来个渲染器设定具体大小。

但是目前为止，我们创建了空的scene，renderer和camera，但是还没有需要渲染的东西，接下来添加坐标轴和平面

var axes=new THREE.AxisHelper(20);
scene.add(axes);

var planeGeometry=new THREE.PlaneGeometry(60,20);
var planeMaterial=new THREE.MeshBasicMaterial({color:0xcccccc});
var plane=new THREE.Mesh(planeGeometry,planeMaterial);
plane.rotation.x=-0.5*Math.PI;

plane.position.x=15;
plane.position.y=0;
plane.position.z=0;

scene.add(plane);

这里创建了一个坐标轴对象(axes),并且通过scene.add方法把坐标轴对象添加到场景中
然后又创建了面板几何对象(planeGeometry),里面参数表示width60，height20，现在大小有了，我们需要给这个面板对象一个外观，就需要创建一个面板材质对象(planeMaterial,我自己暂且这么叫)，参数里面目前写了一个颜色，然后把这两者合二为一，嘭，经过Mesh(网格)，这个搅拌机，就创建出这个面板对象了，接下来的plane.rotation.x=-0.5*Math.PI;是为他初始化位置，后面同理创建了一个小球对象，并且把它添加到scene中。

接下来是本事例最后一部分：

camera.position.x=-30;
camera.position.y=40;
camera.position.z=30;
camera.lookAt(scene.position);

document.querySelector('#WebGL-output').append(renderer.domElement);
renderer.render(scene,camera);

现在所有的需要渲染的物体都已经添加到场景中了，之前说道，相机定义了场景中的哪些东西被渲染。代码中先设定了相机的位置，为了确保相机能拍到这些物体，使用了lookAt函数指向场景的中心。剩下的就是将renderer的输出挂在html的div中。
